# Rock Paper Scissors
Cool to see you here! This is Rock Paper Scissors in Python 3. Exactly what it says on the tin.

### Running instructions
Requires pytest: install using `pip install pytest`.

Run using `python -m rps`.
Test using `python -m pytest`.
